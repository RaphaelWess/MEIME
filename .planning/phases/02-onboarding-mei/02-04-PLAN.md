<objective>
Profile edit surface: a new EmpresaEditPage accessible from ContaTab, allowing the MEI to update atividade_principal and data_abertura_mei after onboarding.

Purpose: D-03 requires the MEI to be able to edit their company profile from the Conta tab. Without this plan, empresa_mei data is write-once at onboarding, which is a UX dead end if the user made a mistake. This plan delivers the edit flow and updates ContaTab to display the current empresa_mei data.

Output: ContaTab with empresa_mei display section and edit button, EmpresaEditPage with pre-filled editable form, and /app/conta/empresa route registered in App.tsx.
</objective>
