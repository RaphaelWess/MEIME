Create src/utils/faturamento.ts as a pure utility module (no default export, JSDoc on every export, no side effects).

Imports: only type imports — import type { EmpresaMei } from '@/services/empresa.service' and import type { Transacao } from '@/services/transacao.service'.

Export three integer centavo constants with numeric separator for readability: LIMITE_MEI_PADRAO = 8_100_000, LIMITE_CAMINHONEIRO = 25_160_000, THRESHOLD_DESENQUADRAMENTO = 9_720_000.

Export two types: ProjecaoResult (discriminated union with tipo: 'hidden' | 'dentro_do_limite' | 'mes_ano'; the 'mes_ano' variant includes mes: number and ano: number) and AlertaNivel (union literal type: 70 | 90 | 100 | 97200).

Export calcLimiteAnual(empresa: EmpresaMei | null, currentYear: number): number
- If empresa is null or empresa.data_abertura_mei is null → return limiteBase (LIMITE_CAMINHONEIRO if is_caminhoneiro, else LIMITE_MEI_PADRAO)
- Parse data_abertura_mei using getUTCFullYear() and getUTCMonth() on new Date(empresa.data_abertura_mei) — this avoids timezone offset bugs (Pitfall 1 in RESEARCH.md)
- If aberturaYear < currentYear → return limiteBase
- If aberturaYear === currentYear: mesAbertura = getUTCMonth() + 1 (1-indexed), mesesRestantes = 12 - (mesAbertura - 1), return Math.round((mesesRestantes / 12) * limiteBase) — Math.round avoids float precision issues (Pitfall 3)

Export calcTotalFaturado(transacoes: Transacao[]): number — reduce sum of t.valor over all transacoes (all are already tipo='entrada' from getByYear)

Export calcPercentual(totalFaturado: number, limiteAnual: number): number — if limiteAnual === 0 return 0; else return (totalFaturado / limiteAnual) * 100. Result can exceed 100.

Export calcProjecao(totalFaturado: number, limiteAnual: number, currentYear: number, today: Date = new Date()): ProjecaoResult
- mesesDecorridos = today.getMonth() (0-indexed month = count of complete months, per Pitfall 5)
- if mesesDecorridos < 1 → return { tipo: 'hidden' }
- mediaFaturamentoMensal = Math.round(totalFaturado / mesesDecorridos)
- if mediaFaturamentoMensal === 0 → return { tipo: 'hidden' }
- limiteRestante = limiteAnual - totalFaturado
- if limiteRestante <= 0 → return { tipo: 'mes_ano', mes: today.getMonth() + 1, ano: currentYear }
- mesesParaEstourar = Math.ceil(limiteRestante / mediaFaturamentoMensal)
- targetMonth0 = today.getMonth() + mesesParaEstourar (0-indexed addition)
- targetYear = currentYear + Math.floor(targetMonth0 / 12)
- targetMonth = (targetMonth0 % 12) + 1 (back to 1-indexed)
- if targetYear > currentYear → return { tipo: 'dentro_do_limite' }
- return { tipo: 'mes_ano', mes: targetMonth, ano: targetYear }

Export calcAlertaAtivo(totalFaturado: number, limiteAnual: number, percentual: number): AlertaNivel | null
- if totalFaturado >= THRESHOLD_DESENQUADRAMENTO → return 97200 (D-10 hierarchy: R$97.200 most severe)
- if percentual >= 100 → return 100
- if percentual >= 90 → return 90
- if percentual >= 70 → return 70
- return null
