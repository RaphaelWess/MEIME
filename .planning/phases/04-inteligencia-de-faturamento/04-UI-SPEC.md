## Color

Source: `src/index.css` — brand tokens override shadcn defaults at bottom of file.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | oklch(1 0 0) = white (#FFFFFF) | Page background, card backgrounds, gauge track background |
| Secondary (30%) | oklch(0.97 0 0) ≈ #F5F5F5 | Card surfaces (`rounded-xl border p-4`), gauge card background |
| Accent (10%) | #16A34A (green-600 equivalent) | Gauge progress fill (0–69% safe zone), positive values (Entradas) |
| Destructive | #DC2626 (red-600) | Alert at 100% threshold, metric card negative values (Saídas) |

Accent reserved for: gauge bar fill in safe range (0–69%), "Entradas" metric value color. Nothing else.

### Alert Color Map (from CONTEXT.md D-11)

These are semantic overrides applied only inside alert card components. They are NOT general accent colors.

| Threshold | Background | Border / Left stripe | Text |
|-----------|-----------|----------------------|------|
| 70% | yellow-50 | yellow-400 | yellow-800 |
| 90% | orange-50 | orange-400 | orange-800 |
| 100% | red-50 | red-400 | red-700 |
| R$97.200 | red-100 | red-700 | red-900 |

Gauge fill color by threshold (matches alert severity):
- 0–69%: `#16A34A` (accent green)
- 70–89%: `#EAB308` (yellow-500)
- 90–99%: `#F97316` (orange-500)
- 100%+: `#DC2626` (red-600)
- R$97.200+: `#991B1B` (red-800)

## FaturamentoGauge — Visual Spec

Source: CONTEXT.md D-02, D-14, D-15, D-16

```
┌─────────────────────────────────────────────┐
│  Limite Anual                  Caminhoneiro? │  ← heading row; "Caminhoneiro" badge
│                                              │     only if is_caminhoneiro=true
│  [████████████░░░░░░░░░░░░░░░░░░░░░░]  55%  │  ← progress bar, full-width, h-3
│                                              │
│  R$ 44.550 de R$ 81.000 consumidos          │  ← body text (text-sm, muted-foreground)
│                                              │
│  Projeção: você atinge o limite em Set/2026  │  ← text-sm; hidden if D-15 conditions met
└─────────────────────────────────────────────┘
```

- Progress bar: `<div>` with `rounded-full` track + inner fill div. Height: 12px (h-3). No external chart library.
- Fill transitions: `transition-all duration-500 ease-in-out` CSS only. No JS animation libraries.
- Percentage: `text-lg font-semibold` right-aligned in the bar row.
- Caminhoneiro badge: `text-xs font-medium bg-blue-100 text-blue-700 rounded px-2 py-0.5` — shown inline next to "Limite Anual" heading only when `is_caminhoneiro=true`.
- Empty/skeleton state: when `empresa` is null OR data is loading, render 2 Skeleton blocks (h-5 for bar, h-4 for text line). Never show zero values.

## FaturamentoAlert — Visual Spec

Source: CONTEXT.md D-09, D-10, D-11, D-12, D-13

- Only the **most severe** active alert is rendered (D-10). One alert card maximum.
- Not dismissable. No close button. Persistent while threshold is active.
- Left border stripe: 4px solid, color per threshold table above.
- Structure: `rounded-xl border-l-4 p-4 flex flex-col gap-2`

```
┌╠══════════════════════════════════════════════╗
│║  [Icon] Atenção: 90% do limite consumido.    ║
│║  Considere desacelerar as receitas até o     ║
│║  final do ano.                               ║
│║                                              ║ ← only for R$97.200 alert:
│║  [Saiba como se desenquadrar →]              ║
╚╩══════════════════════════════════════════════╝
```

## Interaction Contract

| Interaction | Behavior |
|-------------|----------|
| Page load | InicioTab renders Skeleton for gauge + alert area while `useFaturamento` loads; 4 metric cards skeleton unchanged |
| Data loaded, no threshold crossed | FaturamentoGauge renders; FaturamentoAlert is null (not rendered) |
| Data loaded, threshold crossed | FaturamentoAlert renders between gauge and metric grid; only most severe alert shown |
| "Saiba como se desenquadrar" tap | Opens `https://www.gov.br/empresas-e-negocios/pt-br/empreendedor` in new tab; no in-app navigation |
| empresa is null (no onboarding) | FaturamentoGauge renders skeleton variant — never shows R$0 / 0% as real data |
| Month navigation (selectedYear changes) | `useFaturamento(selectedYear)` refetches; gauge and alert update; staleTime: 0 |

## Accessibility

- Alert cards use `role="alert"` so screen readers announce threshold crossing
- Progress bar uses `role="progressbar"` with `aria-valuenow={percentual}` `aria-valuemin={0}` `aria-valuemax={100}` `aria-label="Faturamento anual consumido"`
- "Saiba como se desenquadrar" link includes `aria-label="Saiba como se desenquadrar do MEI (abre em nova aba)"` to signal new-tab behavior
- Minimum touch target for the CTA button: 44px height
