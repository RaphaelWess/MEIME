Create src/components/FaturamentoGauge.tsx. Props-only presentational component — no internal hook calls, no Supabase imports.

Imports:
- Skeleton from '@/components/ui/skeleton'
- centsToBRL from '@/utils/currency'
- MONTHS_PT from '@/stores/financas.store'
- type ProjecaoResult from '@/utils/faturamento'
- THRESHOLD_DESENQUADRAMENTO from '@/utils/faturamento' (needed by getGaugeColor)

Define FaturamentoGaugeProps interface with: totalFaturado: number, limiteAnual: number, percentual: number, projecao: ProjecaoResult, isCaminhoneiro: boolean, isLoading: boolean.

Define getGaugeColor(percentual: number, totalFaturado: number): string helper function (not exported) inside the file:
- if totalFaturado >= THRESHOLD_DESENQUADRAMENTO → return '#991B1B'
- if percentual >= 100 → return '#DC2626'
- if percentual >= 90 → return '#F97316'
- if percentual >= 70 → return '#EAB308'
- return '#16A34A'

The component renders:

LOADING STATE (when isLoading is true): a card div (className "rounded-xl border p-4 flex flex-col gap-3") containing two Skeleton elements: Skeleton with className "h-5 rounded" (for the bar), Skeleton with className "h-4 rounded w-3/4" (for the text line). Never show zero values while loading.

LOADED STATE: a card div (className "rounded-xl border p-4 flex flex-col gap-3") containing:

1. Heading row: div with justify-between items-center flex. Left side: span "Limite Anual" (text-sm font-medium text-zinc-700). Right side: only if isCaminhoneiro is true, render a span badge "Caminhoneiro" with className "text-xs font-medium bg-blue-100 text-blue-700 rounded px-2 py-0.5".

2. Bar row: div with flex items-center gap-3. Contains the progress bar div and the percentage text.
   - Progress bar wrapper: div with role="progressbar", aria-valuenow={Math.min(percentual, 100)}, aria-valuemin={0}, aria-valuemax={100}, aria-label="Faturamento anual consumido", className="flex-1 bg-zinc-100 rounded-full h-3 overflow-hidden"
   - Inner fill div: className="h-full rounded-full transition-all duration-500 ease-in-out", style={{ width: Math.min(percentual, 100) + '%', backgroundColor: getGaugeColor(percentual, totalFaturado) }}
   - Percentage text: span with className "text-lg font-semibold text-zinc-900 shrink-0", content: Math.round(percentual) + '%'

3. Consumed line: p element with className "text-sm text-zinc-500". Content: centsToBRL(totalFaturado) + ' de ' + centsToBRL(limiteAnual) + ' consumidos'. All values via centsToBRL — all values as JSX text nodes, never via raw HTML injection (T-04 pattern from InicioTab).

4. Projection line (conditional): only render when projecao.tipo is not 'hidden'.
   - When projecao.tipo === 'dentro_do_limite': p element (text-sm text-zinc-500) with text "Na projeção atual, você termina o ano dentro do limite." (D-16)
   - When projecao.tipo === 'mes_ano': p element (text-sm text-zinc-500) with text "Projeção: você atinge o limite em " + MONTHS_PT[projecao.mes - 1] + '/' + projecao.ano. MONTHS_PT index is 0-based, projecao.mes is 1-based.
   - When projecao.tipo === 'hidden': render nothing (null) — no placeholder text (per UI-SPEC copywriting contract)

Default export: export default FaturamentoGauge.

All text content rendered as JSX text nodes only (no raw HTML injection via the prop whose name combines "dangerously" and "SetInnerHTML").
